# 37. DAW 청감 체크리스트 (2026-07-20)

> **목적**: docs/34 전 웨이브와 docs/35 S1–S4가 "운영 게이트"로 미뤄 온 청감 검증의 실행 시트.
> 자동 게이트(골든/널/refit)는 *변화 탐지*와 *자기일관성*을 보증할 뿐, "좋게 들리는가"는 사람 귀의 몫이다.
>
> **방법**: 각 모드 × 소스 조합을 A/B(바이패스 대비, 레벨 매칭 ±0.1dB)로 청취. 항목당 ✅/⚠️/❌ + 한 줄 메모.
> 레벨 매칭은 플러그인의 매칭 기능 또는 LUFS 미터로. 모니터링 볼륨 고정.
>
> **세션 준비**: 48kHz 세션, OS 4×(기본), Realism 기본값, 시드 3개 이상 리롤해 개별성 확인.

## 공통 점검 (모든 모드)

| # | 항목 | 확인 방법 |
|---|---|---|
| 공1 | 로드 직후 팝/썸프 없음 | 무음 상태 로드, 첫 2초 청취 (워밍업 thump는 의도된 미세 현상 — 클릭/팝과 구별) |
| 공2 | 파라미터 스윕 클릭 없음 | Drive/Output/Mix 자동화 스윕 |
| 공3 | 프리셋/AB 전환 연속성 | 재생 중 전환 반복 |
| 공4 | 바이패스 투명성 | 널 테스트 모드 또는 위상 반전 합 |
| 공5 | 리롤 개별성 | 시드 3개 — "다른 유닛, 같은 기종"으로 들리는가 |
| 공6 | Mains 50/60 전환 | 험이 들리는 게인에서 라인 주파수 이동 확인 |

## 모드별 × 소스별

각 셀: **톤**(주파수 밸런스 훼손 없음) / **다이내믹스**(펌핑·과도한 압축 없음) / **배음**(음악적, 거칠지 않음) / **노이즈**(음악적 레벨).

### V72 (프리앰프 컬러)
| 소스 | 기대 캐릭터 | 판정 |
|---|---|---|
| 드럼 버스 | 트랜지언트 라운딩, 미드 포워드, 과도한 LF 부풀림 없음 | |
| 보컬 | 실키한 상단, 치찰음 강조 없음, 근접감 | |
| 마스터 | Drive 낮게: 접착감만, 스테레오 이미지 보존 | |

### Console Output (AB급 PP + NFB)
| 소스 | 기대 캐릭터 | 판정 |
|---|---|---|
| 드럼 버스 | 펀치 유지, 클리핑 니 경질감(C9 실루프) 자연스러움 | |
| 보컬 | Drive 2.5+에서 크런치 진입의 음악성 | |
| 마스터 | Realism 스윕: NFB 조임 ↔ 개방감 트레이드오프 | |

### Culture Vulture (디스토션)
| 소스 | 기대 캐릭터 | 판정 |
|---|---|---|
| 드럼 버스 | T/P1/P2 보이싱 차이 명확, 스크린 기아 텍스처(D-A 수정 후 재확인 필수) | |
| 보컬 | BIAS 노브 스윕(−2.0…+1.5V) 전 구간 음악적 사용 가능 | |
| 마스터 | 저레벨에서도 캐릭터 잔존(수정 전 −57dBFS 침묵 문제 재발 없음) | |

### RNDI (DI 컬러)
| 소스 | 기대 캐릭터 | 판정 |
|---|---|---|
| 베이스 DI | LF 새추레이션 밀도, 어택 보존 | |
| 신스 | 디지털 엣지 완화, 고역 손실 없음 | |
| 마스터 | 미묘함 — 있는 듯 없는 듯, 끄면 아쉬움 | |

### HiFi 300B (SE 마스터링)
| 소스 | 기대 캐릭터 | 판정 |
|---|---|---|
| 어쿠스틱 | 짝수 배음 광택, 리액티브 부하(W7)의 저역 탄력 | |
| 재즈/클래식 마스터 | 무NFB 개방감, 과도한 H2 번짐 없음 | |
| 시드 간 게인 산포 | ±4dB 산포(널 게이트 확인치)가 청감상 수용 가능한가 | |

## S2 수정 사항 집중 청취 (이번 사이클 신규)

| # | 항목 | 배경 |
|---|---|---|
| 신1 | CV 전 시드 안정성 | D-A/D-B 수정 — 코너 시드(777 등)에서 폭주/래치 소멸 확인 |
| 신2 | CV 동작점 변화 | 스크린 평형 교정으로 전 유닛 보이싱 이동 — refit이 흡수했는지 귀로 확정 |
| 신3 | Console Realism | C9: 엔벌로프→실루프 전환 — 압축 복원감(레벨 상승) 자연스러운가 |
| 신4 | 워밍업 thump | 전 모드 로드 직후: 물리적 현상이되 과도하지 않은가 (특히 SE/OPT 모드) |
| 신5 | 펜토드 Rg 곡률 | C2: V72 하드 드라이브 H2 증가 — 음악적인가 |
